Node getaran menggunakan Arduino Nano sebagai pengontrol dan nRF24L01 sebagai modul komunikasi data. Node ini dilengkapi dengan sensor getaran piezoelektrik untuk mendeteksi getaran dan sensor kelembaban tanah untuk mengukur kelembaban. Data dari kedua sensor ini dikirim ke node lingkungan melalui nRF24L01. Node getaran ditenagai oleh panel surya kecil dan baterai 18650, sehingga dapat beroperasi secara nirkabel.
