Gambar menunjukkan hubungan komunikasi data antar node. Node 2 mencatat data kelembaban dan suhu tanah, kemudian mengirim permintaan penyiraman ke Node 3 yang berfungsi sebagai pompa irigasi. Node 1, sebagai penyedia sumber air utama, menerima permintaan air dari Node 3 jika pasokan air di waduk irigasi menurun. Node 3 juga mengirim data aliran air ke Node 5, bersama dengan data dari Node 2. Node 4 mencatat listrik yang dihasilkan panel surya dan listrik yang digunakan dari baterai, yang menjalankan pompa utama dan katup solenoid di Node 1. Node 4 mengirim data langsung ke Node 5 untuk diunggah ke internet, dengan rekap data dibagi antara Node 3 dan 4 karena keterbatasan paket data pada nRF24L01
