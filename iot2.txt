Node lingkungan berfungsi untuk mengumpulkan data dari sensor cahaya (BH1750), suhu dan kelembaban udara (DHT11), serta suhu tanah (DS18B20). Data dari node lingkungan, bersama dengan data dari node getaran yang diterima melalui nRF24L01, diunggah ke internet menggunakan modul GSM SIM800L. Node ini menggunakan panel surya dan baterai 18650 untuk daya, memungkinkan operasi tanpa tergantung pada WIFI.
